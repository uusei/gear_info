## for NW 
equation set as follow

 ### $i=1+\frac{Z_b Z_c}{Z_a Z_d}$

## for NGW
equation set as follow

 ### $i=1+\frac{Z_b}{Z_a}$ 

which $Z$ means number  of teeth，$i$ means gear ratio
